SITE WEB DU PROJET S301

VOIX CITOYENNE

L'utilisateur est d'abord dirigé vers la page de connexion, puis soit la page d'inscription s'il n'a pas encore de compte, soit la page d'accueil s'il se connecte.

Fonctionnalités principales :
- Connexion et inscription des utilisateurs
- Création, modification et suppression de groupes
- Invitation d'utilisateurs à rejoindre des groupes
- Création, modification et suppression de thèmes
- Gestion des propositions et des votes
- Affichage des notifications
- Gestion des membres des groupes
- Modification des informations de compte utilisateur

Technologies utilisées :
- PHP pour la logique serveur
- MySQL pour la base de données
- PHPMailer pour l'envoi d'emails
- HTML, CSS et JavaScript pour l'interface utilisateur
